    整体说明 General

    布局类型

    仿真产品使用 2 类布局。

    1. 任务型产品，使用12 栅格的布局。包括：
      • 前端框架：UI Framework
      • 仿真平台：NI Simulation - Simulation Tasks
      • 仿真平台：NI Simulation - Simulation Statics
    2. 工具型产品，使用0 栅格的布局。包括：
      • 仿真平台：NI Simulation - Simulation Analyzer
      • 内部工具：Scenario Cloning - Scenario Visualizer（暂缓）

    响应式布局

    • NavInfo Simulation 正式版（第一版），不使用响应式布局。
      • 只做桌面版网页的默认布局。
      • 暂时不做移动端的布局。
    • UI Framework，预研响应式布局。
      • 研发优先做 12 栅格。
      • 产品设计做移动端布局设计。

    界面宽度说明

    12 栅格

    1. margin width = 56
    2. column width = 96
    3. gutter width = 16
    4. default
      • page width = 1440
      • columns = 12

    0 栅格

    1. 顶部、底部固定。
    2. 与顶部、底部、侧边的距离/间距固定。
