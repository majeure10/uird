# UI Framework 产品页面需求

## 页面布局

![UI Framework页面](../imgs/ns_ui_framework/Layout.png)

1. 页面居中显示内容宽为1024px
2. 两侧留白为208px（当页面宽幅为1440px时）
3. 两侧菜单栏宽均为192px，左侧显示一级标题，二级标题...，右侧菜单栏显示最末级标题下的子标题
4. 中间内容显示宽为608px，并与两侧菜单栏相距16px
